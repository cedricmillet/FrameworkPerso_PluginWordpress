=== INSTALLATION ===
- Déplacer le plugin dans le dossier /wp-content/plugins/
- Se rendre sur le panneau d'aministration Wordpress
- Cliquer sur Extensions > Extensions installées
- Activer le plugin



=== DESINSTALLATION ===
- Se rendre sur le panneau d'aministration Wordpress
- Cliquer sur Extensions > Extensions installées
- Cliquer sur "Désactiver"
- Pour supprimer le plugin de façon permanente, cliquer sur "Supprimer"
